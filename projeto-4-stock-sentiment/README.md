# Stock Market Sentiment Analysis

## Objetivo
Analisar o sentimento de mercado para ações brasileiras com base em notícias.

## Ideia
Coletar notícias via SerpAPI para cada ação e classificar como positiva ou negativa.

## Arquitetura
SerpAPI → Python → Oracle → RAW → REFINED

## Análises
- Volume de notícias por ação
- Notícias positivas vs negativas

## Tecnologias
- Python
- Oracle
- SQL
- SerpAPI

## Insight
Permite entender o sentimento do mercado sobre cada ativo.
